# Scripts

## Format

| Offset | Size | Description                |
| ------ | ---- | -------------------------- |
| 0x00   | 0x04 | Signature `LSP\0`          |
| 0x04   | 0x04 | ?                          |
| 0x08   | 0x04 | If bit0, fields 20 and 24  |
| 0x0C   | 0x04 | Offset to command sections |
| 0x10   | 0x04 | Number of commands         |
| 0x14   | 0x04 | ?                          |
| 0x18   | 0x04 | Offset to ?                |
| 0x1C   | 0x04 | Offset to ?                |
| 0x20   | 0x04 | Offset to ?                |
| 0x24   | 0x04 | Offset to ?                |
