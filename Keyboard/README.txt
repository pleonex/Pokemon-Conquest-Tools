Keyboard is a program to modify the keys of the keyboard.
You must open the file "overlay9_12" DECRYPTED, you can decrypt it
using the program "blz.exe" by CUE.
The following values must be set depending of the region.
Tested only with the name keyboard (don't know if it's used in other place).

Pokemon Nobunaga Ambition (JAP):

Base offset: 0x0225DAC8
RAM:         0x022376A0

Pokemon Conquest (USA and EUR):

Base offset: 0x02259B78
RAM:         0x02233C20
